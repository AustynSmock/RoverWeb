Project Description

The purpose of our rover is to provide users with a 3D model, floor map, and Virtual-Reality perspective of the inside of a structure. This is made possible through three methods:

  1. SLAM navigation
  2. Computer Vision path/lane navigation
  3. Manual navigation

The rover collects map data that is simultaneously uploaded and updated to a web user interface. Users can create an account through the web UI to connect to the rover, view and download map data, and control the rover.
