Getting Started

  1. Acquire Yahboom G1-Tank and Raspberry Pi 4B
  2. Assemble G1-Tank with Raspberry Pi
  3. Follow guide exactly to install Ubuntu, ROS, and RealSense: Setting up Raspberry Pi 4 with Ubuntu 20.04, ROS, Intel RealSense
  4. Install ros1-legacy branch from Intel RealSense GitHub
  5. Download RealVNC Server
  6. Connect Raspberry Pi to RealVNC account and connect to Wi-Fi
  7. Download the RowdyNav Folder from GitHub: RowdyNav GitHub Repository
  8. Ensure OpenCV and Ultralytics are installed: pip install opencv-python ultralytics
  9. cd into RowdyNav folder
  10. Run the command: python3 control.py
  11. In a new terminal, run source /opt/ros/noetic/setup.bash
  12. In the same terminal, run roscore
  13. In a new terminal (keep the previous one open), run source /opt/ros/noetic/setup.bash
  14. In the same terminal, run roslaunch realsense2_camera opensource_tracking.launch
  15. Follow control.py instructions for navigation and control
